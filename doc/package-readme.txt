The files delivered alongside this readme are intended for deployment on the webex.com site.

Contents:

- content/dam/wbx/us/images/workspace/

This directory contains all the necessary site assets.  They are expected to be deployed to the directory with the same
absolute path on the webex.com site: '/content/dam/wbx/us/images/workspace/'.

- workspaces/

This directory contains all the HTML pages to be deployed.

- content-for.html-head

This contents of this file must be copied into the HTML <head> section of each of the deployed pages.
